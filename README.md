# proteomics_wt-vs-sert
Proteomics data analysis using Python

## Overview
This project demonstrates a Python-based workflow for proteomics data analysis.

## Workflow
- Differential protein analysis using log2 fold change and t-test
- Volcano plot visualization
- Pathway enrichment analysis using KEGG and GO databases
- Multi-database enrichment visualization using bubble plots

## Tools
- Python
- pandas
- numpy
- scipy
- matplotlib
- GSEApy
